# 🎮 坦克大战 (Tank Battle)

基于 HTML5 Canvas 和原生 JavaScript 开发的经典坦克大战网页游戏。

## 🚀 快速开始

直接用浏览器打开 `tank-battle.html` 即可游玩，无需安装任何依赖。

## 🎯 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格 / J | 射击 |

## 🎁 道具系统

| 道具 | 效果 |
|------|------|
| ⚡ 加速 | 提升坦克移动速度 |
| 🔥 快速射击 | 缩短射击间隔 |
| 💥 多重射击 | 同时发射多方向子弹 |
| 🛡️ 护盾 | 抵挡一次伤害 |

## 🕹️ 游戏机制

- **波次系统**：每波敌人全部消灭后自动进入下一波，敌人数量和强度逐步提升
- **可破坏墙壁**：棕色砖墙可被子弹摧毁，灰色钢墙不可破坏
- **粒子特效**：爆炸、击中、道具拾取均有粒子动画
- **音效**：射击、爆炸、拾取道具等均有音效反馈
- **生命系统**：初始 3 条命，被击中损失一条命并短暂无敌

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6)
- Web Audio API (音效)
- CSS3 (UI 动画)
